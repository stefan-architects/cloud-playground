## 🧰 1. Install or Update AWS CLI
Install or update the AWS CLI using the official AWS documentation:  
🔗 https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html

---

## 🔐 2. Configure Your Credentials
Use the AWS CLI to set up your access keys and default region.

---

## ☁️ AWS CloudShell (Optional but Recommended)
You can run all AWS commands directly in AWS CloudShell without installing anything locally.

### ▶️ Open CloudShell
🔗 https://console.aws.amazon.com/cloudshell/home

### 📦 CloudShell Comes Preinstalled With
• 🚀 AWS CLI v2  
• 🧩 AWS SDK tools  
• 🔐 Pre‑authenticated environment  

💡 Use CloudShell if you prefer not to install the AWS CLI locally.
