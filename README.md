# 🌩️ Cloud Playground  
A mono‑repo for hands‑on AWS and DevOps practice — built on curiosity, repetition, and the occasional “wait… why did that happen?” moment. This is where I test ideas, break things safely, rebuild them cleaner, and slowly turn cloud concepts into muscle memory. Nothing here is perfect; everything here is progress.

## 🧪 Core Areas & Repo Contents  
This repo is a living space for exploring the fundamentals of cloud architecture and the messy, hands‑on work that comes with learning them:

- **Compute:** EC2 experiments, container tests, scaling patterns, and “what happens if I tweak this setting?” moments  
- **Storage:** S3 layouts, lifecycle rules, access patterns, and durability experiments  
- **Security:** IAM least‑privilege attempts, policy debugging, and guardrail testing  
- **Database:** RDS snapshots, parameter tuning, query behavior tests, and small data‑modeling experiments  
- **Networking:** VPC designs, routing puzzles, subnet layouts, and connectivity troubleshooting  
- **DevOps Tooling:** AWS CLI, Bash, Git/GitHub workflows, automation trials that may or may not behave on the first run  
- **Troubleshooting:** Break‑fix practice, behavior testing, CloudWatch deep dives, and “let’s see what happens if…” scenarios  

Inside the repo you’ll find:  
- **Labs** — focused AWS experiments  
- **Prototypes** — early architectural tests  
- **Patterns** — reusable IaC/DevOps building blocks  
- **Notes** — explanations, diagrams, and thought‑process snapshots
