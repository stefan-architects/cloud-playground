# ☁️ Cloud Playground  

This repo is a mix of real‑world scenarios, experiments, half‑formed AWS ideas, and late‑night prototypes. The structure reflects that same energy—loose, exploratory, and intentionally chaotic. Nothing is polished or production‑ready; everything is open for exploration.

---

## 🧩 What You’ll Find Inside
- 🧱 **Single‑Service Deep Dives** — focused explorations of one AWS service
- 🔗 **Multi‑Service Prototypes** — lightweight integrations to see how components interact
- 🧨 **Architecture Behavior Tests** — “what happens if…” scenarios
- 🌀 **Chaos & Curiosity** — ideas that don’t fit anywhere else, including breaking things on purpose

---

## 🚀 Why This Exists

Cloud intuition isn’t built by reading docs — it’s built by tinkering, experimenting, and occasionally yelling “OH THAT’S WHY” at your terminal.

This repo is a celebration of that process.
