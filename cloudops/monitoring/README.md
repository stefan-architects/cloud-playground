## 💡 About this Digital Course
The **Monitoring** module discusses monitoring and logging services to implement metrics, alarms, and filters. Additionally, the course covers how to identify and remediate issues and how to implement performance optimization strategies for database resources.

## 📈 Module Objectives
- Use Amazon CloudWatch to monitor AWS resources.
- Create and use CloudWatch dashboards to visualize information.
- Create sharable custom dashboards.
- Use CloudWatch alarms to monitor environments.
- Implement automated remediation strategies with Amazon EventBridge.
- Use AWS monitoring and logging services to manage CloudWatch log lifecycles.
- Implement CloudWatch agents to monitor and troubleshoot issues.
